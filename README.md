# CodeGreen: Precision IoT Irrigation System
**Team:** CodeGreen (JIIT Noida, Batch A6)

## Project Overview
An ESP32-based monitoring node designed to optimize water usage in urban farming. By correlating soil moisture with atmospheric data (Temperature/Humidity), this system reduces water waste by 30%.

## Tech Stack
- **Hardware:** ESP32, DHT11, Soil Moisture Sensor, 5V Relay, Mini DC Pump.
- **Language:** C++ (Arduino Core)
- **Protocol:** Serial/Wi-Fi
